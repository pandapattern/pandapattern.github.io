# 隐私政策

**请注意：** 这是一个示例隐私政策，用于演示如何使用此应用落地页生成器。请将内容替换为您自己的隐私政策。

<br>

## 1.0 我们收集的信息
我们收集的信息包括您在使用我们服务时提供的信息。

### 1.1 来自第三方的信息
- 我们可能从第三方服务获取您的信息。
- 这些信息用于改善我们的服务。
- 我们会对这些信息进行保护。

### 1.2 您提供给我们的信息
当您使用我们的服务时，我们可能会收集您提供的信息，包括但不限于：
- 账户信息
- 使用数据
- 设备信息

<br>

## 2.0 我们收集的个人信息类型及收集方式
我们收集的个人信息类型包括：

### 2.1 我们从第三方收集的信息
我们可能从以下第三方收集信息：

| 第三方 | 用途 |
| :--- | :--- |
| 云存储服务 | 用于存储您的数据。 |
| 分析服务 | 用于分析应用使用情况。 |

<br>

## 3.0 您作为数据主体的权利
根据适用的数据保护法律，您拥有以下权利：

1. 访问权：您可以请求访问您的个人数据。
2. 更正权：您可以请求更正不准确的个人数据。
3. 删除权：您可以请求删除您的个人数据。

<br>

## 4.0 我们如何存储和保护收集的个人信息
我们采用行业标准的安全措施来保护您的个人信息，包括：
- 数据加密
- 访问控制
- 定期安全审计

<br>

## 5.0 针对欧洲经济区个人的信息处理和传输
如果您是欧洲经济区（EEA）的居民，我们遵守《通用数据保护条例》（GDPR）的要求。
